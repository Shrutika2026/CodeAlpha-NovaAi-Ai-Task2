# ✨ Nova AI: Seismic Neural Interface
> **Advanced Technical Support Interface for the Multi-Modal AI Seismic Forecasting (MASF) Framework.**

---

## 🛰️ Project Vision
**Nova AI** is a specialized NLP (Natural Language Processing) engine built to bridge the gap between complex seismic forecasting data and user inquiries. Developed as a core internship project in 2026, this tool ensures that information regarding earthquake preparedness and the MASF methodology is accessible, accurate, and visually engaging.

## 🧠 Core Intelligence (The Brain)
To satisfy technical requirements for high-precision matching, Nova AI bypasses simple keyword searches in favor of **Mathematical Contextual Awareness**:

* **Preprocessing:** Utilizes the `NLTK` library for tokenization and lemmatization. It strips away "noise" words and focuses on the root meaning of every query.
* **Vectorization (TF-IDF):** Converts human language into a *Term Frequency-Inverse Document Frequency* matrix. Essentially, it turns words into coordinates in a multi-dimensional mathematical space.
* **Cosine Similarity:** The engine calculates the cosine of the angle between the user's input vector and the pre-defined knowledge base to find the closest logical match.



## 🎨 Visual Identity & UI
The interface follows a **"Cyber-Luxe"** aesthetic designed for a professional "Mission Control" environment:
* **Animated Neons:** A high-performance gradient background flowing through **Purple, Blue, and Pink**, representing seismic energy waves.
* **Glassmorphism:** Semi-transparent containers with `backdrop-filter` effects for a modern, sleek look.
* **High-Readability Mode:** Contrast-optimized text fields (Dark-on-White) to ensure the interface is accessible and easy to navigate.
* **Real-time Feedback:** A dynamic **Signal Strength** bar that visualizes the AI's confidence for every response.

## 🛠️ Technical Stack
* **Language:** Python 3.10+
* **Frontend:** Streamlit (Custom CSS Injection)
* **Machine Learning:** Scikit-Learn (Linear Models)
* **NLP Library:** NLTK (Natural Language Toolkit)
* **Data Structure:** JSON (Modular Knowledge Base)
